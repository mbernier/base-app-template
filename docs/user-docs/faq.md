# Frequently Asked Questions

## General

### What is [App Name]?

[App Name] is a web application built on Base, an Ethereum Layer 2 network. It allows you to [describe your app's purpose].

### Do I need cryptocurrency to use [App Name]?

For basic features like browsing collections, no. For minting and transactions, you'll need ETH on the Base network. If you use Coinbase Smart Wallet, some transactions may be sponsored (free) for you.

### What is Base?

Base is a secure, low-cost Ethereum Layer 2 network built by Coinbase. Transactions on Base are much cheaper and faster than on Ethereum mainnet while maintaining the same security guarantees.

## Wallets

### Which wallets are supported?

- **Coinbase Wallet** (recommended) - includes Smart Wallet for gasless transactions
- **MetaMask**
- Most WalletConnect-compatible wallets

### What is Smart Wallet?

Coinbase Smart Wallet is a wallet that lets you transact without paying gas fees. The app sponsors your transaction costs. It's the easiest way to get started if you're new to crypto.

### How do I add ETH to Base?

1. Bridge ETH from Ethereum mainnet using the [Base Bridge](https://bridge.base.org)
2. Purchase directly on Base through Coinbase
3. Transfer from another wallet that has ETH on Base

## NFTs and Minting

### What does "minting" mean?

Minting is the process of creating a new copy of a digital asset (NFT or token) on the blockchain. When you mint, you receive a unique token in your wallet.

### How much does minting cost?

Costs vary by collection. Each collection page shows the mint price. On Base, gas fees are typically under $0.01.

### Where do my minted NFTs go?

Minted tokens go directly to your connected wallet address. You can view them in [App Name] or in any wallet that supports Base network tokens.

### What are the different token types?

- **ERC-721**: Unique, one-of-a-kind tokens (traditional NFTs)
- **ERC-1155**: Tokens that can have multiple copies of the same item
- **ERC-20**: Fungible tokens (like coins) where each unit is interchangeable

## Account and Security

### How does authentication work?

[App Name] uses SIWE (Sign-In With Ethereum). Instead of a username and password, you sign a message with your wallet to prove you own the address. No password is stored.

### Is my data safe?

- We never have access to your private keys
- Authentication is done via cryptographic signatures
- Your wallet connection can be revoked at any time by disconnecting

### How do I disconnect my wallet?

Click your wallet address in the top right corner and select **Disconnect** or **Sign Out**.

## Troubleshooting

### The page is loading slowly

- Check your internet connection
- Try refreshing the page
- Clear your browser cache

### A transaction is stuck

- Base transactions typically confirm in 2-3 seconds
- If stuck, check the transaction on [BaseScan](https://basescan.org)
- Try refreshing and attempting the transaction again
